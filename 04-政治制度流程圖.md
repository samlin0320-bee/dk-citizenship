## 【1. 法案立法流程 Lovgivningsprocessen】

```
[ 提案 Fremsættelse ]
   政府部長或議員提出草案
              ↓
[ 一讀 1. behandling ]
   Folketinget 進行原則性辯論
              ↓
[ 委員會審查 Udvalg ]
   專責委員會詳細討論、聽取專家/民眾意見、撰寫報告
              ↓
[ 二讀 2. behandling ]
   逐條審查並可提出修正案
              ↓
[ 三讀 3. behandling ]
   最終辯論與表決
   ⚠️ 需過半數（至少 90 名議員出席才有效）
              ↓
[ 御筆簽署 Kongelig stadfæstelse ]
   國王 + 負責部長共同簽署
              ↓
[ 公告生效 Træde i kraft ]
   正式成為法律
```

### 考點
- **三讀制度** 與 **委員會審查** 之間的位置
- **kongelig stadfæstelse** 由「國王 + 部長」雙簽
- 表決需 ≥ 90 名議員出席（179 席過半門檻）

## 【2. 三層政府架構】

```
┌─────────────────────────────────────────┐
│         國家 STATEN                       │
│  - 國防 (Forsvar)                        │
│  - 警察 (Politi)                          │
│  - 司法/法院 (Domstole)                   │
│  - 外交 (Udenrigspolitik)                 │
│  - 高等與高中教育                          │
│  - 稅收規則                                │
└────────────────┬────────────────────────┘
                 ├──────────────────────┐
                 ↓                      ↓
┌────────────────────────┐ ┌─────────────────────────┐
│  大區 5 REGIONER        │ │  市鎮 98 KOMMUNER        │
│  - 公立醫院             │ │  - 托嬰/幼兒園            │
│  - 醫療保健系統          │ │    (Vuggestuer/Børnehaver)│
│    (Sundhedsvæsen)     │ │  - 中小學 (Folkeskole)    │
│                        │ │  - 長照 (Ældrepleje)      │
│  ⚠️ Region 沒有徵稅權   │ │  - 就業中心 (Jobcentre)   │
│  ⚠️ 靠國家撥款運作      │ │  - 社福與難民整合         │
│                        │ │  ⚠️ Kommune 有徵稅權      │
└────────────────────────┘ └─────────────────────────┘
```

### 考點速記
- **Region 主管「醫院」** 一字就夠
- **Kommune 主管「貼近民眾的服務」**（小孩、老人、學校、工作）
- **徵稅權**：只有 Staten + Kommune 有，**Region 沒有**
- **Region 是 2007 年**改革時設立（取代原本的 amt）

## 【3. Folketinget 組成與運作】

### 組成
```
總席次：179 席
├─ 丹麥本土 175 席
├─ 格陵蘭 (Grønland) 2 席
└─ 法羅群島 (Færøerne) 2 席
```

### 選制
- **比例代表制** Forholdstalsmetoden
- **政黨門檻 (Spærregrænse)：2%**（北歐最低之一）
- 因此多黨林立，少數政府是常態

### 議會制原則 Parlamentarisme
```
        ┌──────────────────────────┐
        │  選民 Vælgerne 投票          │
        └────────────┬─────────────┘
                     ↓
        ┌──────────────────────────┐
        │  Folketinget 179 席         │
        └────────────┬─────────────┘
                     ↓
        ┌──────────────────────────┐
        │  Regering 政府              │
        │  ⚠️ 不能有 Folketing 多數反對  │
        │  ⚠️ 即使少數派也可組閣        │
        └──────────────────────────┘
```

### 主要政黨光譜
```
                      ← 紅營 ───── 中間 ───── 藍營 →

紅綠聯盟  替代黨  社會  社民黨  激進自由黨  溫和黨  自由黨  保守  自由聯盟  丹麥民主黨  丹麥人民黨
  Ø        Å    人民黨   A         B        M       V      C       I         Æ          O
                 F
```

### 考點
- **2% 門檻** 是常考題
- **少數政府合法** 是常考題
- **政府部長不一定要是議員**
- **首相隨時可解散國會** 提前大選

## 【4. 三權分立 Magtens tredeling】

```
       [ 立法權 Lovgivende magt ]
       由 Folketinget + Regering 共用
                  ↑
                  │
                  │ 質詢、不信任投票、彈劾法院 Rigsretten
                  │
                  ↓
       [ 行政權 Udøvende magt ]
       由 Regering (政府) 行使
                  ↑
                  │
                  │ 違憲審查、糾正違法處分
                  │
                  ↓
       [ 司法權 Dømmende magt ]
       由 Domstolene (獨立法院) 行使
```

### 互相制衡機制
- **法院** → 審查法律是否違憲、糾正行政機關違法處分
- **議會** → 質詢、**不信任投票 (Mistillidsvotum)** 逼部長下台
- **議會** → 將違法部長送 **彈劾法院 Rigsretten**

### 考點
- **誰行使哪個權力** 是必背
- **Rigsretten 是彈劾法院**，不是普通司法
- **Højesteret 是最高法院**（普通司法的頂端）

## 【5. 公民權利與義務 Rettigheder og pligter】

### 憲法保障的核心自由
```
Grundlov 保障的權利
│
├─ 言論自由 Ytringsfrihed
│     └─ ⚠️ 禁止事前審查 (Censur forbudt)
├─ 集會自由 Forsamlingsfrihed
├─ 結社自由 Foreningsfrihed (組黨、工會、社團)
├─ 宗教自由 Religionsfrihed
├─ 財產權 Ejendomsret
└─ 住宅不可侵犯 Boligens ukrænkelighed
```

### 公民義務
```
Borgerlige pligter
│
├─ 納稅 Betale skat
├─ 義務役 Værnepligt
│     └─ ⚠️ 18 歲以上具國籍之男女皆有
└─ 公民義務職 Borgerligt ombud
      └─ 被抽中須擔任 國民陪審員 Lægdommer
```

### 申訴管道
```
公民認為政府機關違法或疏失
              ↓
[ 議會監察使 Folketingets Ombudsmand ]
   ⚠️ 獨立於政府之外
   ⚠️ 由 Folketinget 任命
   ⚠️ 只監督公共行政機關
```

### 考點
- **Ombudsmand** 是常考題（不是 Etisk Råd、不是 Pressenævnet）
- **義務役男女皆需**（2024 後新法）
- **言論自由 = 禁事前審查**，不等於沒有事後追訴

## 【6. 國家教會 Folkekirken】

```
┌──────────────────────────────────────────────┐
│   丹麥福音路德派國家教會                          │
│   Den evangelisk-lutherske folkekirke            │
├──────────────────────────────────────────────┤
│                                              │
│   國教地位                                    │
│   ├─ 憲法特殊保障                              │
│   ├─ 國家財政支持                              │
│   └─ ⚠️ 國王必須是其成員                       │
│                                              │
│   政教關係                                    │
│   ├─ ⚠️ 無單一最高宗教領袖 (沒有「主教長」)      │
│   └─ 政府的宗教事務部 Kirkeministeriet 統管      │
│                                              │
│   公民身分 vs 教會成員                         │
│   ├─ 加入完全自願                              │
│   ├─ 公民可不入、退教或信他教                   │
│   └─ ⚠️ 教會稅 Kirkeskat 只有成員才繳          │
│                                              │
└──────────────────────────────────────────────┘
```

### 考點（極常考）
1. **「丹麥公民必須加入 Folkekirken 嗎？」→ Nej（不需要）**
2. **教會稅只有成員繳**，非公民義務
3. **國王必須是 Folkekirken 成員**（憲法規定）
4. **沒有「最高宗教領袖」**，行政由 Kirkeministeriet 管
